# tabs_management_crx
a google chrome extension to organize my tabs

# how to use it :
1. download the folder with all the files.
2. go to the google chrome settings -> extensions
3. toggle developper mode
4. click "load unpackaged extension"
5. select the folder that you've downloaded and the extension will be activate.

# the extension allows you to :
1. copy a list of all your opened tabs
2. paste a list in a text field to open all your tabs in a row
3. manage quickly your opened tabs
